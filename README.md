# `aquiletour2021`

## Commandes

1. Pour démarrer la version JavaFX

        $ ./gradlew javafx

1. Pour compiler vers javascript

        $ ./gradlew js

1. Pour démarrer le serveur:

        $ ./gradlew server
        
    * visiter <a href="http://localhost:8080">http://localhost:8080</a>
    * <kbd>Entrée</kbd> pour arrêter le serveur
